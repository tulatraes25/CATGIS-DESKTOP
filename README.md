# CATGIS Desktop

Aplicacion de escritorio Java para visualizacion y edicion de informacion geoespacial.

## Requisitos

- Java 21
- Gradle

## Ejecucion

```bash
gradle run
```

## Estructura

- `src/ar/com/catgis`: codigo fuente Java
- `src/icons`: iconos SVG
- `src/main/resources/icons`: recursos PNG
