3. Soil Grids ISRIC (WCS)
4. 5 escenarios de riesgo (derrame, inundación, deslizamiento, incendio, contaminación)
5. Geomorfones (11 landforms desde DEM)

7. Simulación de inundación (DEM + lluvia + escorrentía)
8. CATSERVER stack completo
9. 28 templates cartográficos curados
10. Import .qpt QGIS
11. Module system toggleable (11 módulos)
12. Analysis Console unificada
13. Quick Style Panel dockeado
14. Welcome Page profesional
15. Map decorations live
16. Feature decorators
17. Gradient fills con LRU cache
18. DXF export SOLID
19. Label expression engine (34 funciones)
